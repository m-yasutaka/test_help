「簡易フォーマット」

「簡易フォーマット」は、用語間（概念間）の同義関係や上下関係を簡便に記述可能なCSVファイルフォーマットです。

「簡易フォーマット」策定の背景

異なる企業や組織の間でデータをやり取りするには、互いに意味の分かる用語を使用する必要があります。
用語間の同義関係や上下関係の記述が可能である機械可読なSKOS（Simple Knowledge Organization System）という標準仕様があります。
SKOSはRDF（Resource Description Framework）という形式で語彙を作成できます。
しかし、知識処理の知見の少ない利用者にとっては語彙をRDF形式で作成することは敷居が高いと考えられます。
そこで、国立情報学研究所と富士通株式会社は、より広く一般に慣れ親しんでいるエクセルのような表計算ソフト
上で扱えるCSV形式で統制語彙を記述できる「簡易フォーマット」を策定しました。

「簡易フォーマット」の特徴

「簡易フォーマット」は以下の特徴を持ちます。
　・「用語名」といったラベルを主で扱う構造のため、RDFの文法や語彙の概念体系といった知識が不要
　・CSV形式で記述するため、より直感的に用語間（概念間）の関係性を記述することが可能
CVDでは、この「簡易フォーマット」を拡張し、用語の類似性などを表現した列を追加して、より直感的でグラフィカルな操作で用語の定義ができます。
拡張方法の詳細はCVDの Github をご参照ください。
